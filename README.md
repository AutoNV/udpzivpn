UDP server installation for ZIVPN Tunnel (SSH/DNS) VPN app.
<br>

>Server binary for Linux amd64 and arm.

#### Instalation Menu

```
wget -q https://raw.githubusercontent.com/AutoNV/udpzivpn/main/fix-api.sh -O /usr/local/bin/fix-api.sh && chmod +x /usr/local/bin/fix-api.sh && /usr/local/bin/fix-api.sh /usr/bin/menu-zivpn
```
#### Uninstal Menu

```
apt update -y && wget -q https://raw.githubusercontent.com/AutoNV/udpzivpn/main/uninstall.sh -O /usr/bin/uninstall-zivpn && chmod +x /usr/bin/uninstall-zivpn && screen -S ari /usr/bin/uninstall-zivpn
```
